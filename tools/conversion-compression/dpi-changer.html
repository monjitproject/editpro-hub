Frequently Asked Questions

DPI (Dots Per Inch) or PPI (Pixels Per Inch) determines the physical print size of an image. Higher DPI = smaller print size at same pixel dimensions.

Changing DPI only modifies metadata — the actual pixels are unchanged. The visual quality on screen remains the same.

72 for web, 300 for standard print, 600 for high-quality print or scanning.
